# Premium Budget Tracker

A polished personal finance dashboard built with React, Vite, and Chart.js.

## Features
- Add, edit, and delete income/expense transactions
- Monthly summaries and charts
- CSV import/export backup
- Responsive dashboard experience

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy
- GitHub Pages: publish the dist folder from the build
- Netlify: use the included netlify.toml
- Render: create a static site and set the build command to `npm run build`
